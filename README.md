Voting System

A secure and user-friendly Online Voting System built using HTML, CSS, and JavaScript, featuring both User Voting Panel and Admin Dashboard.

 Features
 User Panel:-
 
User Registration & Login
Secure Voting System (one user = one vote)
View Candidates/Options
Real-time vote status display
Logout functionality

Admin Panel:-

Admin Login Authentication
Add / Edit / Delete Candidates
View All Users
Monitor Voting Results
Reset Voting Data (if implemented)
Dashboard overview

Tech Stack :--

Frontend: HTML, CSS, JavaScript
using local storage
